#　拓殖大学工学部情報工学科
![alt text](image-1.png)
　情報化社会をさらに**進化**させる、IT業界の新たな**リーダー**を育成
##　３コース＋コラボ２コース
* システムエンジニアリングコース
  * プログラミングによって時代の求める新しい情報システムを作る
* インタネット&セキュリティコース
  * 安全で安心できる持続的な情報社会を支える*ネットワーク*とセキュリティを学ぶ
* 知能情報コース
  * 「モノ」と「コト」の未来へAIの先を拓く

1. ビジュアルコンピューティングコース
   コンピュータを用いて、様々な事象や現象をわかりやすく可視化・表現するための技術を学ぶ
1. Webデザインコース
　ユーザとクライアントを繋ぐ新しいWebの価値を創造する
